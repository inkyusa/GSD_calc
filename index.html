Ground Sample Distance Calculator

This page contains a calculator for determining the pixel size of a given camera sensor and the ground sample distance for that sensor given pixel size, sensor size, focal length, and altitude.

Pixel Size Calculator

Ground Sample Distance

The Ground Sample Distance is a measure of resolution in aerial photogrammetry. It measures the distance between pixel centers as measured on the ground.

1. Photo Footprint Area returns zero after calculating GSD when values for image height and image width are missing. Area is calculated as (Image Width * GSD) * (Image Height * GSD).
